
This is the backend service for the AgriTech platform that connects **farmers** and **dealers**.  
It allows dealers to add products and farmers to place orders, along with authentication and real-time features.

---

## 🚀 Features

- 🔐 User Authentication (JWT + OTP via Email)
- 👨‍🌾 Farmer & Dealer Roles
- 📦 Dealer Product Management
- 🛒 Farmer Order System
- 🔔 Notifications (Planned / In Progress)
- ⚡ Real-time Auction System (Socket.io)
- 🌐 RESTful APIs

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- Socket.io
- Nodemailer (for OTP/email)
